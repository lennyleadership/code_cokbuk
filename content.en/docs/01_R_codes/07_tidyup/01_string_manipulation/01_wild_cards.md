# Wild card `^`
It indicates the following letter is the first letter of the string.

# Wild card `.*`
The example is `^stp_process.*\\.xlsx`

# Wild card `\\`
Add it before `.`.  The example is: `^stp_process.*\\.xlsx`

Add it before `$`.  The example is: `\\$ICPMS-GWMP_`
